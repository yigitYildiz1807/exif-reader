# exif-reader
Fotoğrafın çekildiği GPS konumunu ve tarih-saat bilgilerini alabilirsiniz

# Kurulum
Tek yapmanız gereken terminalinizde şu kodu çalıştırmak:

```
pip install exifread==2.3.2
```

# Kullanım
```
python reader.py
```
Komutu ile programı başlatabilirsiniz. Okuyacağınız fotoğrafın reader.py dosyası ile aynı klasör içinde olması önemlidir. Programa fotoğrafın ismini doğru bir şekilde uzantısı ile girdikten sonra entera basarsanız program size çıktıyı hemen verecektir.
